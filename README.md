# CampusTrack Lost and Found Portal

## Features
### Student
- Register/Login
- Post lost/found items
- View all items
- Search items
- View my posts

### Admin
- Register/Login
- View all users/items
- Delete posts
- Block/unblock users

## Tech Stack
- Backend: Java Spring Boot + MongoDB + JWT
- Frontend: React + Tailwind + Axios

## Run Backend
cd backend
mvn spring-boot:run

## Run Frontend
cd frontend
npm install
npm start
